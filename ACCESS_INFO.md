## 📱 手机端访问步骤

1. **获取电脑IP地址**
   ```powershell
   ipconfig | findstr "IPv4"
   ```

2. **确保手机和电脑在同一网络**

3. **在手机浏览器中输入**
   ```
   http://[电脑IP]:8000
   ```

## 🚀 快速启动命令

```powershell
# 启动HTTP服务器
python -m http.server 8000

# 启动实时同步
.\sync-realtime.ps1 -Interval 30

# 检查系统状态
.\check-errors.ps1
```

## 📞 故障排除

### 如果无法访问
1. 检查防火墙设置
2. 确认端口8000未被占用
3. 验证网络连接

### 如果同步失败
1. 检查Git配置
2. 验证GitHub权限
3. 查看同步日志
